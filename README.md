# PoC RSS Loader for OpenCTI (Linux)

Автоматический загрузчик Proof-of-Concept файлов из GitHub репозиториев в платформу OpenCTI через RSS-ленту.

## Функциональность

### RSS мониторинг
- Чтение RSS-ленты с PoC репозиториями
- Настраиваемый интервал проверки (по умолчанию 5 минут)
- Bootstrap-режим для обработки последних N элементов при старте
- Непрерывный мониторинг в реальном времени

### Обработка репозиториев
- Клонирование GitHub репозиториев через git clone
- Извлечение CVE ID из заголовков RSS
- Фильтрация файлов по расширениям, именам, директориям и размеру
- Временное хранение в рабочей директории с автоматической очисткой

### Создание объектов OpenCTI
- **Artifact** - загрузка PoC файлов с метаданными и хешами
- **Tool** - создание инструмента для каждого PoC с версией
- **Vulnerability** - автоматическое создание уязвимости по CVE ID
- **Identity** - создание автора репозитория
- **External Reference** - ссылки на GitHub репозиторий и NVD
- **Relationships** - связывание всех созданных объектов

### Интеграции
- **NVD API** - получение данных о CVE (описание, CVSS, метрики)
- **GitHub** - извлечение версии инструмента из файлов конфигурации
- **OpenCTI** - GraphQL API для создания объектов и связей

### Управление данными
- Кеширование обработанных репозиториев в JSON
- Предотвращение повторной обработки
- Создание архивов репозиториев для прикрепления к Tool
- Детальное логирование с ротацией

## Режимы работы

- **Непрерывный мониторинг** - постоянная проверка RSS-ленты
- **Однократная проверка** - разовая обработка текущих элементов
- **Bootstrap-режим** - обработка последних N элементов перед переходом в realtime
- **Тестовый режим** - проверка интеграций и зависимостей

## Требования

- Linux система с Python 3.8+
- Git
- Доступ к OpenCTI платформе
- Доступ к интернету

## Установка и управление

Для установки и управления контейнером используйте команды из файла [DOCKER_COMMANDS.md](DOCKER_COMMANDS.md).

## Конфигурация

Настройка через `config.yml`:
- Подключение к OpenCTI
- RSS-источник
- Интервалы мониторинга
- Правила фильтрации файлов
- Настройки логирования
- Ограничения по размеру файлов
